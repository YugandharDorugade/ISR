üß© 1Ô∏è‚É£ Background: What is an API?
An API is a communication bridge between two software systems.
It allows a program to request information from another service.
The service sends back data in structured format (usually JSON).

üìò Example:
When you open a weather app, it calls an API like
api.openweathermap.org to fetch live data for your city.

‚öôÔ∏è 2Ô∏è‚É£ OpenWeatherMap API
We‚Äôll use the OpenWeatherMap API ‚Äî a free and popular weather service.
Base URL:
https://api.openweathermap.org/data/2.5/weather?q={CITY}&appid={API_KEY}&units=metric
Parameter	Meaning
q		City name (e.g., Pune, Mumbai)
appid		Your unique API key
units		Metric ‚Üí Celsius (optional)

üí° 3Ô∏è‚É£ How It Works
User inputs city name.
Python script sends an HTTP GET request to the weather API.
API returns a JSON object containing all details.
Script extracts and displays required fields.

üíª 5Ô∏è‚É£ Python Code Implementation
import requests  # Library to make web requests

# Step 1: User input
city = input("Enter city name: ")

# Step 2: API details
api_key = "YOUR_API_KEY_HERE"  # (You can get a free one from openweathermap.org)
base_url = "https://api.openweathermap.org/data/2.5/weather"

# Step 3: Make API call
url = f"{base_url}?q={city}&appid={api_key}&units=metric"
response = requests.get(url)
data = response.json()  # Convert JSON to Python dictionary

# Step 4: Extract and print data
if data["cod"] == 200:
    print(f"\nWeather Report for {data['name']}:")
    print("Temperature:", data["main"]["temp"], "¬∞C")
    print("Humidity:", data["main"]["humidity"], "%")
    print("Pressure:", data["main"]["pressure"], "hPa")
    print("Wind Speed:", data["wind"]["speed"], "m/s")
    print("Description:", data["weather"][0]["description"])
else:
    print("City not found or API error.")
