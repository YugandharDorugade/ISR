6ï¸âƒ£ Algorithm (From Manual + Simplified)
# Step 1: Input text documents (already conflated)
docs = {
    "D1": "this is a text",
    "D2": "a text has words",
    "D3": "words are made from letters"
}

# Step 2: Build inverted index
index = {}
for doc, text in docs.items():
    for word in text.split():
        if word not in index:
            index[word] = set()
        index[word].add(doc)

# Step 3: Display index
print("Inverted Index:")
for term, occ in index.items():
    print(term, "â†’", occ)

# Step 4: Query search
query = input("Enter query: ").split()
results = [index.get(q, set()) for q in query]
final = set.intersection(*results)
print("Documents retrieved:", final)

ğŸ’¬ Short Answer Questions
Q1. Working of Inverted Files
Ans:
An inverted file maps each term (from the vocabulary) to a list of documents where it occurs (occurrence list).
Searching is done by looking up the query term in the index and retrieving all associated documents.

Q2. Applications of Inverted Index
Ans:
Used in:
Search engines
Document retrieval systems
Content-based filtering
Text mining

Q3. Working of Signature Files
Ans:
Each document is represented by a bit string (signature).
Each bit represents whether a term from a dictionary appears in the document.
Query signatures are compared with document signatures using bitwise operations â€” faster but may produce false hits.

ğŸ¤ Viva Questions (With Short Answers)
No.	Question	Short Answer
1	What is vocabulary and occurrences?	Vocabulary = list of unique terms; Occurrences = list of document IDs where each term appears.
2	How is search carried out on inverted index?	By finding query terms in the vocabulary, fetching their occurrence lists, and combining them (AND/OR).
3	How to index multimedia objects?	By extracting features (e.g., color histograms, shape descriptors, audio fingerprints) and indexing them like keywords.
4	Limitations of inverted index?	High storage overhead, frequent updates are costly, unsuitable for dynamic databases.
5	What is suffix array and suffix tree?	Both store suffixes of text for fast substring search; suffix tree is hierarchical, suffix array is space-efficient linear version.
6	What is the concept of signature files?	Signature = bit string representation of document terms; used to speed up approximate search.
