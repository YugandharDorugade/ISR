üß© Background Context
Images are not directly comparable like text or numbers ‚Äî we need to describe them using quantitative values.
For example, a ‚Äúred rose‚Äù and ‚Äúred apple‚Äù share a color feature (red), but differ in shape and texture.
So, before comparing or retrieving images, we convert them into feature vectors ‚Äî a compact numerical representation.
This process is called Feature Extraction.

Features are measurable properties that capture essential visual information about an image.

Statistical features	Derived using math formulas	Mean, variance, standard deviation

üßÆ 3Ô∏è‚É£ Mathematical Representation (for Texture)
The GLCM (Gray-Level Co-occurrence Matrix) is the most important method.
It‚Äôs a matrix where element (i,j) represents how often a pixel with intensity i is adjacent to a pixel with intensity j.
It depends on:
Distance (d) between pixels
Angle (Œ∏): 0¬∞, 45¬∞, 90¬∞, 135¬∞
From this, we derive features like contrast, energy, etc.

‚öôÔ∏è 4Ô∏è‚É£ Step-by-Step Implementation (Python)
Let‚Äôs use cv2 (OpenCV), skimage.feature, and matplotlib.
import cv2
import numpy as np
from matplotlib import pyplot as plt
from skimage.feature import greycomatrix, greycoprops

# Step 1: Read image
img = cv2.imread('flower.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# Step 2: Plot color histogram
plt.figure(figsize=(10,4))
color = ('b','g','r')
for i,col in enumerate(color):
    hist = cv2.calcHist([img],[i],None,[256],[0,256])
    plt.plot(hist, color = col)
plt.title("Color Histogram")
plt.xlabel("Pixel Intensity")
plt.ylabel("Frequency")
plt.show()

# Step 3: Compute GLCM matrix (distance=1, angle=0)
glcm = greycomatrix(gray, [1], [0], 256, symmetric=True, normed=True)

# Step 4: Extract texture features
contrast = greycoprops(glcm, 'contrast')[0,0]
energy = greycoprops(glcm, 'energy')[0,0]
homogeneity = greycoprops(glcm, 'homogeneity')[0,0]

print("Contrast:", contrast)
print("Energy:", energy)
print("Homogeneity:", homogeneity)

üé§ 7Ô∏è‚É£ Viva Questions (With Detailed Answers)
No.	Question	Answer
1	What is feature extraction?	Process of converting raw image data into numerical descriptors (features) that represent important properties like color, texture, or shape.
2	Why is feature extraction necessary?	Because raw images are high-dimensional and hard to compare directly. Features make similarity search and classification computationally efficient.
3	What are color features?	Features that represent distribution of colors in the image, typically through histograms in RGB or HSV space.
4	Define texture features.	Describe spatial arrangement of pixel intensities (patterns of smoothness or roughness) using statistical measures like contrast, entropy, energy.
5	What is GLCM?	Gray-Level Co-occurrence Matrix ‚Äî represents frequency of intensity pairs (i, j) in an image at a specific distance and direction.
6	What are GLCM-based features?	Contrast, Energy, Homogeneity, Correlation, Entropy.
7	Difference between energy and entropy.	Energy = uniformity (lower randomness), Entropy = randomness (higher disorder).
8	What is a color histogram?	Graph showing frequency of pixel intensities for each color channel (R, G, B).
9	What is shape feature extraction?	Describes geometric properties like object boundaries, area, and perimeter.
10	Application of feature extraction?	Image classification, object detection, pattern recognition, CBIR.
